# Affan Ahmed Portfolio

Personal portfolio for Affan Ahmed, an aspiring full stack developer and cloud computing enthusiast based in Karachi, Pakistan.

## What is included

- Responsive Next.js portfolio homepage
- Profile image and downloadable CV in `public/`
- Sections for about, experience, skills, projects, education, and contact
- Project links connected to Affan's public GitHub repositories
- Contact form that opens the visitor's email client with a prepared message

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Main content files

- `utils/data/personal-data.js`
- `utils/data/projects-data.js`
- `utils/data/experience.js`
- `utils/data/educations.js`
- `utils/data/skills.js`
