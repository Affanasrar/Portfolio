This is my personal portfolio website. I built it to showcase who I am, what I am learning, and the projects I have worked on as an aspiring full stack developer, web technologies enthusiast, and cloud computing learner based in Karachi, Pakistan.

## About Me

I am Affan Ahmed, an undergraduate Computer Science student at Salim Habib University. I am focused on building responsive, user-friendly web applications and improving my skills across frontend development, backend development, databases, cloud deployment, and practical software engineering.

Alongside my studies, I work as a Web Development Instructor at Shams Commercial Institute and have experience as a Full Stack Web Developer Intern at Darul Solution.

## Run Locally

Clone the project, install dependencies, and start the development server.

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build For Production

```bash
npm run build
npm run start
```
